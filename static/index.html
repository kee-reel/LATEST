Заполняй тестовые наборы перед проверкой - если тест упадёт в твоих тестовых наборах, то это не страшно. Если тест упадёт во внутренних наборах, то за каждое третье такое падение баллы за отчёт будут снижены на 0.5 (максимум на 2 балла). Если падение не зависило от тебя (проблемы автотестов), то оно не будет учтено.

Тестовый набор - это одна строка, с набором вводимых с клавиатуры данных. Набор данных разделяется через ";". Например, чтобы сэмулировать ввод чисел "1 2 3" с клавиатуры, надо составить такой тестовый набор: "1;2;3;". Тестовых наборов может быть несколько, чтобы покрыть все возможные краевые случаи -- указывая несколько наборов, пиши их на отдельных строчках.
